## What GSC claims (paper-grade)

The the current framework late-time release (Option 2 / freeze-frame measurement model) is intentionally narrow:

- It specifies an operational measurement model for late-time observables (how clocks/rods/energies are interpreted in the mapping).
- It treats SN absolute magnitude (`Delta M`) and BAO `r_d` as nuisance parameters in the late-time harness (no first-principles claim at v11.0.0).
- It keeps **redshift drift** as a historical/supporting no-go diagnostic: the sign pattern at `z ~ 2-5` remains useful for specific late-time `H(z)` classes but is no longer the primary project discriminator in Roadmap v2.8.
