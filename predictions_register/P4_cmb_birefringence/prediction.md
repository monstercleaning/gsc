# Prediction P4 — CMB cosmic birefringence

## Statement

The σ-F̃F coupling generated by the gravitational fixed point's renormalization of the dimension-4 topological operator (Section 4 of GSC_Framework.md) predicts a non-zero cosmic birefringence angle:

```
β = ∫_0^{z_CMB} (σ(z) / f_σ) · c · (dt/dz) dz
```

where σ(t) is fixed by the late-time fit (Paper A) and f_σ is the σ-photon coupling derived in Paper B.

The current Planck hint is β = 0.35° ± 0.14° (Minami & Komatsu 2020). GSC predicts a specific amplitude consistent with this hint, given the late-time σ-fit.

## Tier

**T3** — tests both the RG-induced σ-F̃F coupling (Paper B) and the late-time σ-fit (Paper A). Falsification eliminates the σ-axion-equivalence claim.

## Scoring algorithm

For each released measurement β_obs with uncertainty σ_β:

```
z = (β_obs - β_predicted) / sqrt(σ_obs^2 + σ_pred^2)
```

Pass if |z| < 3 at all release stages.

The Planck consistency is a *bonus* test (can score now); the LiteBIRD measurement is the **decisive** test (precision factor ~7 over Planck).
