## White-Hat Outreach Policy

Outreach must stay white-hat: no paid links, no link exchanges, no keyword-stuffed guest posts,
and no manipulative ranking tactics. For search policy alignment, follow Google link spam policy

## Citation Guidance (5 lines)

1. Prefer the archival DOI once minted (`{{ZENODO_DOI}}`).
2. If DOI is pending, cite the shipped release tag.
3. If tag is unavailable in context, cite the repository URL and commit hash.
4. Use `CITATION.cff` metadata as canonical for software citation fields.
5. Keep legacy `v10*` identifiers as provenance only, not as the primary citation target.
