# GSC Phase 2 (v0.1): From an Action to Redshift-Drift Observables

**Purpose.** GSC v10.1 used a phenomenological power law `H(z)=H0(1+z)^p` (with `p=0.5`)
to illustrate the *sign* difference in redshift drift relative to ΛCDM.
This Phase 2 note closes the “action-to-observable gap” by showing that the same power-law
family arises as an attractor solution in a minimal scalar-field action.

This is **not yet** the full derivation from the GSC RG-running / Landau-pole mechanism.
Instead, it provides a *theoretically standard* bridge: a well-defined action → equations
of motion → `H(z)` → drift.

---

## 1. Minimal action prototype

A canonical scalar field with an exponential potential:

- **Action (Einstein frame)**  
  \[
    S = \int d^4x\sqrt{-g}\left[\frac{M_P^2}{2}R - \frac{1}{2}(\partial\varphi)^2 - V_0 e^{-\lambda\varphi/M_P}\right].
  \]

Such models admit **power-law attractor solutions** when the scalar dominates the energy density.

---

## 2. Power-law solution and mapping to `H(z)`

For scalar domination, the scale factor evolves as:

\[
  a(t) \propto t^{q}, \quad q = \frac{2}{\lambda^2}.
\]

From this:

\[
  H(z) = H_0(1+z)^{p}, \quad p = \frac{1}{q} = \frac{\lambda^2}{2}.
\]

So the Phase 1 choice `p=0.5` corresponds to `λ=1`.

**Interpretation.** The exponent is no longer a “magic number”. It is the slope of a scalar potential
and can be treated as a parameter to be constrained (or derived from a deeper RG mechanism).

---

## 3. Redshift drift (sign)

Using the standard drift relation:

\[
  \dot z = H_0(1+z) - H(z),
\]

we have for the power-law family:

\[
  \dot z > 0 \;\;\forall z>0 \quad \Longleftrightarrow \quad p<1.
\]

Thus **any** action choice yielding `p<1` produces an always-positive drift, in contrast to ΛCDM
(which predicts a sign change at `z~2–3`).

---

## 4. Distance–drift diagnostic

A key issue for “always positive drift” models is *distance consistency* with ΛCDM:
forcing `H(z) < H0(1+z)` at high redshift tends to reduce the comoving distance integral.

This package includes a simple diagnostic that compares:

- `H(z)/H0` for ΛCDM vs a power-law model (default `p=0.5`)
- distance modulus residuals `Δμ(z)` relative to ΛCDM
- velocity drift `dv/dt0` (cm/s/yr)

and a second diagnostic scanning a mild **transition ansatz** `p(z)` to show the tension
between matching luminosity distances and enforcing positive drift.

## 6. Limitations and next steps

1. **RG origin not yet implemented.** The exponential potential is a standard proxy.
   Next: derive or approximate an effective `V(φ)` from the GSC RG-running hypothesis.
2. **Matter/radiation not included in the action solver.** The present prototype is scalar-dominated.
   Next: include additional components and test whether “always positive drift” can persist
   while still matching early-time observables.
3. **CMB / Boltzmann evolution not included.** A full alternative to ΛCDM requires a consistent
   CMB power spectrum computation (Phase 3).
